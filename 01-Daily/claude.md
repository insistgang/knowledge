# 01-Daily 每日复盘模块

> 多层次复盘系统：日 → 周 → 月 → 季度 → 年度

## 文件结构

```
01-Daily/
├── 2026-02/              # 按月存放日记
├── ChatHistory/          # Claude 聊天记录
└── Templates/            # 复盘模板
```

## 复盘触发词

| 指令 | 行为 |
|------|------|
| 今日复盘 / 写日记 | 日复盘（读取 `今日复盘skill.md`） |
| 本周复盘 / 周复盘 | 周复盘 |
| 本月复盘 / 月复盘 | 月度复盘 |
| 季度复盘 | 季度复盘 |
| 年度复盘 | 年度复盘 |

## 日复盘流程

1. 读取昨天的「明日待办」
2. 让用户自由说今天做了什么
3. 追问遗漏细节（时间线、成果、卡点）
4. 补充：状态评分、习惯打卡、感悟、明日计划
5. 生成 markdown 日记，保存并同步 Hexo

## 周复盘维度

1. 本周概览（2-3句话）
2. 关键成果（3-5项）
3. 习惯追踪（论文阅读、散步等）
4. 模式识别（精力波动、卡点）
5. 未完成事项分析
6. 本周感悟
7. 下周调整
