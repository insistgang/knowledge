# Attachments 附件存放区

> 图片、PDF、文件

## 文件结构

```
Attachments/
├── images/            # 图片文件
├── pdfs/              # PDF 文档
└── files/             # 其他文件
```

## 使用规则

### images/
- 论文图表截图
- 实验结果图片
- 思维导图

### pdfs/
- 论文全文
- 参考文档
- 技术手册

### files/
- 数据文件
- 代码备份
- 压缩包

## 引用方式

```markdown
![图片描述](Attachments/images/图片名.png)
[PDF名称](Attachments/pdfs/文件名.pdf)
```

## 注意事项

- 定期清理无用附件
- 文件命名要有意义
- 大文件 (>10MB) 谨慎存放
