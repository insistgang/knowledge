# 提示词助手模块

> 存放各类 AI 助手的提示词模板

## 工作目录
`/mnt/e/000/knowledge/提示词助手`

## 可用助手

| 文件名 | 功能 |
|--------|------|
| `体育生命题专家.md` | 体育理论命题助手 |
| `体重管理管.md` | 体重管理顾问 |
| `小程序修复cc提示词助手.md` | 小程序 bug 修复助手 |
| `美赛建模智能体.md` | 美赛数学建模助手 |

## 提示词模板结构

每个提示词文件应包含：

1. **角色定义** - 你是谁，有什么专长
2. **任务范围** - 能做什么，不能做什么
3. **工作流程** - 如何执行任务
4. **输出格式** - 结果应该长什么样
5. **注意事项** - 特殊约束或偏好

## 使用方式

当需要使用某个助手时：
1. 读取对应的 `.md` 文件
2. 将内容作为 system prompt 注入
3. 开始对话

## 新增助手流程

1. 在当前目录创建新的 `xxx助手.md`
2. 按照模板结构编写提示词
3. 更新本文件的"可用助手"表格
