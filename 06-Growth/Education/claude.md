<answer_templates>
【主观题答题模板】

**材料分析题（301/302）**
```
总：该教师的做法是[正确/错误]的，[符合/违背]了[XX理念/法规/原则]。

分：
第一，[理论点1]。材料中，[结合材料分析]。
第二，[理论点2]。材料中，[结合材料分析]。
第三，[理论点3]。材料中，[结合材料分析]。

总：因此，作为教师，应该[正面建议/启示]。
```

**辨析题（302）**
```
该说法是[正确/错误/片面]的。

[概念界定]：XX是指...

[分析论证]：
（正确类）因为...,所以该说法正确。
（错误类）但实际上...,所以该说法错误。
（片面类）但该说法忽略了...,所以是片面的。

综上所述，该说法是[正确/错误/片面]的。
```

**简答题（302）**
```
[题目问什么，第一句先定义/总述]

具体包括以下几点：
（1）...
（2）...
（3）...
（4）...
```
