## 一、中文核心期刊引用要求

### 1.1 计算机学报 (Journal of Computer Research and Development)

| 项目 | 要求 |
|------|------|
| **编码方式** | 顺序编码制 [1][2]... |
| **文献数量** | 研究性论文一般不超过15条，综述可调整 |
| **排序规则** | 按文中引用顺序排列 |
| **引用位置** | 正文中引用部位用上标标注 |
| **基本格式** | `[序号] 作者. 文章题目[J]. 刊名, 年份, 卷数(期数): 起止页码.` |

**示例：**
```
[1] Redmon J, Divvala S, Girshick R, et al. You Only Look Once: Unified, Real-Time Object Detection[C]//Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition. 2016: 779-788.
```

### 1.2 软件学报 (Journal of Software)

| 项目 | 要求 |
|------|------|
| **编码方式** | 顺序编码制 |
| **文献数量** | 只选择最主要的文献列入 |
| **排序规则** | 按文中出现先后顺序编排 |
| **基本格式** | 同计算机学报 |

### 1.3 中国图象图形学报 (Journal of Image and Graphics)

| 项目 | 要求 |
|------|------|
| **编码方式** | 作者-年份制 (Author, Year) |
| **文献数量** | **不少于15篇** |
| **时间要求** | **近10年文献占比≥60%** |
| **排序规则** | 按英文字母顺序排序 |
| **特殊要求** | 中文文献需英中对照，需提供DOI |

**示例（作者-年份制）：**
```
Redmon J, Divvala S, Girshick R, et al., 2016. You Only Look Once: Unified, Real-Time Object Detection. In: Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition (CVPR). 779-788. DOI: 10.1109/CVPR.2016.91
```

## 二、GB/T 7714-2015 标准规范

### 2.1 文献类型标识符

| 标识 | 文献类型 |
|------|---------|
| [M] | 专著 - Monograph |
| [J] | 期刊文章 - Journal |
| [C] | 会议论文集 - Conference |
| [D] | 学位论文 - Dissertation |
| [P] | 专利 - Patent |
| [S] | 标准 - Standard |
| [R] | 报告 - Report |
| [N] | 报纸 - Newspaper |
| [EB/OL] | 电子资源 - Electronic Bulletin/Online |

### 2.2 核心文献格式模板

#### 期刊论文 [J]
```
[序号] 作者. 文章题目[J]. 期刊名, 年, 卷(期): 起止页码.
```
**示例：**
```
[1] He K, Zhang X, Ren S, et al. Deep Residual Learning for Image Recognition[C]//Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition. 2016: 770-778.
```

#### 会议论文 [C]
```
[序号] 作者. 论文题目[C]//会议名称. 出版地: 出版者, 年份: 起止页码.
```

#### 学位论文 [D]
```
[序号] 作者. 论文题名[D]. 保存地: 保存单位, 年份.
```

#### 电子资源 [EB/OL]
```
[序号] 作者. 题名[EB/OL]. (更新日期)[引用日期]. 获取和访问路径.
```

### 2.3 作者著录规则

- **个人作者**：姓前名后，外文姓全大写，名缩写（略去缩写点）
- **三人以内**：全部列出
- **三人以上**：列出前三位，后加", 等"或", et al."
- **作者间**：用逗号分隔

## 四、图表规范

### 4.1 中文图注格式

```
图1 本文提出的多尺度检测网络架构
Fig.1 Architecture of the proposed multi-scale detection network.
```

**要求：**
- 图注位于图下方居中
- 中英文对照
- 图内文字使用中文，缩写需注释

### 4.2 三线表格式

```
表1 不同检测器在COCO数据集上的性能对比
Tab.1 Performance comparison of different detectors on COCO dataset

┌─────────────┬─────────┬─────────┬─────────┐
│  Method     │  AP     │  AP50   │  APs    │
├─────────────┼─────────┼─────────┼─────────┤
│  Faster R-CNN│  42.0   │  62.1   │  24.2   │
│  YOLOX      │  49.7   │  67.6   │  29.5   │
│  Ours       │  51.3   │  69.2   │  32.1   │
└─────────────┴─────────┴─────────┴─────────┘
```

**要求：**
- 表头置于表上方
- 三线表（顶线、底线、栏目线）
- 单位放在栏目名称后
- 中英文表题对照

### 4.3 矢量图要求

| 要求 | 规格 |
|------|------|
| **格式** | .ai, .eps, .svg, .pdf |
| **分辨率** | ≥300 dpi（位图转换后） |
| **字体** | 中文用宋体/黑体，英文用Arial/Times |
| **线条** | ≥0.5 pt，避免过细 |
| **配色** | 彩图需考虑黑白打印效果 |

### 4.4 图片分辨率要求

| 用途 | 最低分辨率 |
|------|-----------|
| 线图/图表 | ≥300 dpi |
| 照片/图像 | ≥300 dpi |
| 组合图 | ≥600 dpi |

## 五、辅助文档模板

### 5.1 投稿声明

```latex
\documentclass{article}
\usepackage{ctex}

\begin{document}

\section*{投稿声明}

本人（们）郑重声明：

1. 本文系原创性研究成果，未公开发表过。

2. 文中引用他人作品已注明出处。

3. 本文不涉及保密内容和泄密问题。

4. 同意编辑部对本文进行编辑加工。

5. 同意将本文的汇编权、数字化复制权、发行权等授予编辑部。

全体作者签名：
\begin{tabular}{@{}ll@{}}
第一作者：\underline{\hspace{4cm}} & 日期：\underline{\hspace{3cm}} \\
第二作者：\underline{\hspace{4cm}} & 日期：\underline{\hspace{3cm}} \\
\end{tabular}

\end{document}
```

### 5.2 作者贡献声明

```
作者贡献

张三：提出算法框架，设计实验，撰写论文

李四：数据集构建，实验验证

王五：代码实现，结果分析

责任作者：张三
```

### 5.3 利益冲突声明

```
利益冲突声明

全体作者声明：
1. 本研究未接受任何可能影响研究客观性的商业资助
2. 作者之间不存在可能影响研究结果的利益关系
3. 本研究不存在其他需要披露的利益冲突
```

### 5.4 Cover Letter 模板

```latex
\documentclass{letter}
\usepackage{ctex}

\signature{张三}
\address{上海工程技术大学\\电子电气工程学院}

\begin{document}

\begin{letter}{《计算机学报》编辑部}

\opening{尊敬的编辑：}

您好！

本人（们）向贵刊投稿论文《面向违法建筑识别的多尺度图像检测方法研究》。

本文的主要创新点如下：

1. 提出了一种新的多尺度特征融合模块，有效提升了小目标检测精度

2. 设计了自适应anchor机制，降低了超参敏感性

3. 在违建检测数据集上取得了优于现有方法的性能

本文未在其他刊物发表，未一稿多投。全体作者已阅读投稿须知，同意编辑部对本文的处理。

感谢您对本文的关注！

\closing{此致\\敬礼}

\end{letter}

\end{document}
```

## 六、引用质量检查清单

### 6.1 数量与时间分布

- [ ] 参考文献总数 ≥15 篇
- [ ] 近3年文献占比 ≥60%
- [ ] 核心期刊/顶会占比 ≥80%

### 6.2 类型分布

- [ ] 期刊论文 [J]
- [ ] 会议论文 [C]
- [ ] 学位论文 [D]
- [ ] 英文文献占比 ≥50%

### 6.3 完整性检查

- [ ] 所有文献均有完整出处
- [ ] DOI齐全（如期刊要求）
- [ ] 作者人数正确（3人以内全列，3人以上列前3位+等）
- [ ] 页码信息完整

### 6.4 文中引用

- [ ] 顺序编码制：按引用顺序编号
- [ ] 作者-年份制：与列表排序一致
- [ ] 引用位置正确（句末或作者后）
- [ ] 无漏引、错引
