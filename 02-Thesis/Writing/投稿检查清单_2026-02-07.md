# 论文投稿检查清单

> 用于投稿前的最终审查

---

## A. 内容完整性检查

### 必需文件清单
- [ ] 正文（中英文）
- [ ] 摘要（中英文）
- [ ] 关键词（中英文）
- [ ] 参考文献
- [ ] 图表（14张+）
- [ ] 作者简介
- [ ] 基金项目信息

### 章节完整性
- [ ] 引言（背景、相关工作、贡献）
- [ ] 方法（问题定义、网络架构、损失函数）
- [ ] 实验（数据集、设置、结果、分析）
- [ ] 结论与展望

---

## B. 格式规范检查

### 图表规范
- [ ] 图表编号连续（Fig.1, Fig.2...）
- [ ] 图表标题中英双语
- [ ] 图片分辨率≥300dpi
- [ ] 三线表格式正确
- [ ] 图表在正文中被引用

### 参考文献
- [ ] 数量≥30篇
- [ ] 近3年文献占比≥50\%
- [ ] 格式符合GB/T 7714-2015
- [ ] 文中引用与列表一致

### 公式编号
- [ ] 公式编号连续
- [ ] 公式在正文中被引用
- [ ] 数学符号定义清晰

## C. 学术规范检查

### 创新点表述
- [ ] 创新点在摘要中明确
- [ ] 创新点在引言中强调
- [ ] 创新点在方法中实现
- [ ] 创新点在实验中验证

### 实验完备性
- [ ] 与至少5个SOTA方法对比
- [ ] 消融实验验证各模块贡献
- [ ] 在公开数据集上验证（如适用）
- [ ] 统计显著性检验

### 代码与数据
- [ ] 代码开源（如适用）
- [ ] 数据集公开（如适用）
- [ ] 实验可复现
