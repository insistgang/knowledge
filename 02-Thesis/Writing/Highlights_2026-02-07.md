# Highlights（创新点提炼）

> 用于投稿系统的创新点描述，每条不超过85字符

---

## 中文版

1. 提出ABMSF模块，融合CNN局部特征与Transformer全局特征，提升小目标检测精度
2. 新增P2层特征金字塔，增强对32×32像素以下井盖的感知能力
3. 构建7类状态细粒度井盖数据集，填补该领域公开数据集空白

---

## English Version

1. Proposed ABMSF module combining CNN local features and Transformer global features for improved small object detection
2. Added P2 layer feature pyramid to enhance perception of manhole covers below 32×32 pixels
3. Constructed a fine-grained manhole cover dataset with 7 status categories, filling the gap in public datasets
