## 四、格式规范检查（GB/T 7714-2015）

### 4.1 期刊论文格式

```
[序号] 作者. 文章题目[J]. 期刊名, 年, 卷(期): 起止页码.
```

**检查项：**
- [ ] 作者人数正确（3人以内全列，3人以上列前3位+等/et al）
- [ ] 期刊名完整
- [ ] 年份准确
- [ ] 卷号(期号)格式正确
- [ ] 起止页码完整

**示例：**
```
❌ 错误：He K. Deep Residual Learning. CVPR, 2016
✓ 正确：He K, Zhang X, Ren S, et al. Deep Residual Learning for Image Recognition[C]//Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition. 2016: 770-778.
```

### 4.2 会议论文格式

```
[序号] 作者. 文章题目[C]//会议名称. 出版地: 出版社, 年份: 起止页码.
```

**检查项：**
- [ ] 会议名完整（不要缩写，如CVPR应写全称）
- [ ] 出版地、出版社信息完整
- [ ] 页码范围准确

### 4.3 电子资源格式

```
[序号] 作者. 题名[EB/OL]. (更新日期)[引用日期]. 获取和访问路径.
```

**示例：**
```
Jocher G, Chaurasia A, Qiu J. Ultralytics YOLOv5[EB/OL]. (2020-07-23)[2024-02-07]. https://github.com/ultralytics/ultralytics.
```

### 4.4 作者著录检查

| 问题 | 正确 | 错误 |
|------|------|------|
| 姓名顺序 | 姓前名后 | 名前姓后 |
| 外文作者姓 | 全大写或首字母大写 | 小写 |
| 外文作者名 | 缩写略去缩写点 | 带缩写点 |
| 三人以上 | 前三位+等/et al. | 全部列出 |
| 作者分隔 | 逗号 | 分号 |

## 五、完整性检查

### 5.1 单条文献检查清单

每条参考文献必须包含：

- [ ] **作者信息**：
  - [ ] 姓名拼写正确
  - [ ] 作者数量正确
  - [ ] 中英文作者名格式统一

- [ ] **文献题名**：
  - [ ] 标题完整（不截断）
  - [ ] 英文标题大小写规范
  - [ ] 中文标题无错别字

- [ ] **来源信息**：
  - [ ] 期刊名/会议名完整
  - [ ] 卷号、期号准确
  - [ ] 出版社信息（会议论文）

- [ ] **时间信息**：
  - [ ] 发表年份准确
  - [ ] 更新日期（电子资源）

- [ ] **位置信息**：
  - [ ] 起止页码
  - [ ] DOI（如期刊要求）
  - [ ] URL（电子资源）

### 5.2 文中引用检查

- [ ] 引用序号连续无跳号

- [ ] 引用位置正确（句末或作者后）

- [ ] 无漏引、错引

## 八、使用说明

### 检查流程

```
1. 导出当前文献列表（从论文中）
   ↓
2. 逐条对照清单检查
   ↓
3. 标记问题条目
   ↓
4. 修正或补充
   ↓
5. 重新检查
   ↓
6. 生成最终引用列表
```

### 优先级

**P0（必须立即修复）：**
- 引用数量不足
- 格式严重错误
- 缺少必引经典论文

**P1（Day 5前完成）：**
- 时间分布不均
- 文献类型不全
- 作者著录错误

**P2（可后续优化）：**
- DOI补充
- 页码完善
- 中英文对照
