# 02-Thesis 论文核心区

> 文献、实验、撰写

## 文件结构

```
02-Thesis/
├── Literature/          # 文献笔记（按日期 dayXX）
├── Experiments/         # 实验记录
└── Writing/             # 论文撰写
```

## 文献管理流程

```
发现论文 → 存入 00-Inbox/
    ↓
阅读时 → 创建 Literature/YYYY-MM-DD-dayXX-论文名.md
    ↓
阅读中 → 填模板，重点写「对我的启发」
    ↓
读完后 → 添加双向链接
```

## 快速指令

- 「读论文」「论文笔记」→ 进入论文阅读模式
- 「文献总结」→ 总结某篇论文核心内容
- 「相关文献」→ 查找相关文献
