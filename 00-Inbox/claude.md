# 00-Inbox 收件箱模块

> 快速捕获区，临时存放待整理的内容

## 工作目录
`/mnt/e/000/knowledge/00-Inbox`

## 用途

- 临时笔记、想法
- 待整理的截图
- 快速记录的灵感
- 待分类的内容

## 工作流程

```
捕获 → 存入Inbox → 每日22:00整理 → 移动到对应文件夹 → 清空Inbox
```

## 文件规则

- 文件名使用日期+关键词：`YYYY-MM-DD-关键词.md`
- 每日复盘时处理 Inbox 中的内容
- 处理完后删除或移动到对应模块

## 整理到何处

| 内容类型 | 目标文件夹 |
|----------|------------|
| 论文相关 | `02-Thesis/` |
| 每日记录 | `01-Daily/` 或 `日记/` |
| 竞赛相关 | `04-Competitions/` 或 `美赛建模分享会/` |
| 技术笔记 | `05-Tech/` 或 `agents code/` |
| 个人成长 | `06-Growth/` 或 `教师资格证备考/`、`记账理财助手/` |
| 资源汇总 | `07-Resources/` |
| 聊天记录 | `claude聊天记录整理/` |
| 提示词模板 | `提示词助手/` |
