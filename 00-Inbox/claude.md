# 00-Inbox 收件箱

> 快速捕获区，临时存放待整理的内容

## 工作流程

```
捕获 → 存入 Inbox → 每日 22:00 整理 → 移动到对应文件夹 → 清空 Inbox
```

## 文件规则

- 文件名：`YYYY-MM-DD-关键词.md`
- 每日复盘时处理 Inbox 中的内容
- 处理完后删除或移动到对应模块

## 整理目标

| 内容类型 | 目标文件夹 |
|----------|------------|
| 论文相关 | `02-Thesis/` |
| 每日记录 | `01-Daily/` |
| 竞赛相关 | `04-Competitions/` |
| 技术笔记 | `05-Tech/` |
| 个人成长 | `06-Growth/` |
| 资源链接 | `07-Resources/` |
| 提示词模板 | `提示词助手/` |
