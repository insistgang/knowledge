AI赋能的农村金融ESG评价系统
===========================

项目概述
-------
本系统是一个"AI赋能的农村金融ESG评价系统"，专为双碳比赛设计。该系统融合了人工智能技术与ESG评价理念，专注于农村地区的可持续发展评估。系统提供智能化的ESG（环境、社会、治理）评价功能、碳资产估值服务、区块链交易记录等功能，支持农村企业、农业合作社等主体的ESG评分、碳资产管理和绿色金融服务。

系统特色
--------
1. AI智能评价
   - 基于机器学习的ESG评分模型
   - 自然语言处理分析企业报告
   - 智能风险识别与预警
   - 自动生成改进建议

2. 农村金融特色
   - 面向农村企业和农业合作社
   - 考虑农业特殊性的评价体系
   - 支持土地经营权、农产品等抵押
   - 绿色农业项目专项评估

3. 多维度ESG评价
   - 环境评分（权重35%）- 考虑土壤、水源、生物多样性
   - 社会评分（权重30%）- 关注乡村振兴、就业带动
   - 治理评分（权重35%）- 强调透明度与合规性
   - 农业行业调整因子
   - 智能评级系统（A+、A、B+、B、C）

4. 碳资产估值服务
   - 农业碳汇项目评估
   - 林业碳汇、农业减排项目
   - 支持多种碳信用类型（CER、CCER、VER、GCER）
   - 动态价格计算
   - 项目年期调整

5. 区块链技术
   - 农产品溯源上链
   - 碳资产交易记录
   - 数据防篡改
   - 智能合约自动执行

6. 大数据支撑
   - 156家农村企业数据
   - 1247条区块链交易记录
   - 187个碳资产项目
   - 2156条ESG评价历史记录

技术架构
--------
后端：Python 3.x + HTTP Server + 机器学习模型
前端：HTML5 + JavaScript + ECharts
AI引擎：自然语言处理 + 机器学习算法
数据格式：JSON
端口：8018

API接口说明
----------
基础URL: http://localhost:8018

1. 系统信息
   GET /health - 获取系统健康状态
   GET /stats - 获取统计数据

2. 企业管理
   GET /api/v1/companies - 获取企业列表（支持分页）
   参数：
     - page: 页码（默认1）
     - page_size: 每页数量（默认50）

3. ESG评价
   POST /api/v1/esg/evaluate - ESG评分
   请求体：
     {
       "company_id": 企业ID,
       "environmental_score": 环境评分,
       "social_score": 社会评分,
       "governance_score": 治理评分
     }

4. 碳资产估值
   POST /api/v1/carbon/valuation - 碳资产估值
   请求体：
     {
       "type": "cer|ccer|ver|gcer",
       "amount": 数量,
       "project_years": 项目年限
     }

5. 区块链交易
   GET /api/v1/blockchain/transactions - 获取交易记录（支持分页）

6. 碳资产管理
   GET /api/v1/carbon-assets - 获取碳资产列表

7. ESG历史
   GET /api/v1/esg/history - 获取ESG评价历史

8. 系统监控
   GET /api/system/status - 系统状态
   GET /api/system/metrics - 性能指标
   GET /api/system/alerts - 告警信息

数据结构
--------
1. 农村企业数据（companies_data.json）
   - id: 企业唯一标识
   - name: 企业/合作社名称
   - industry: 所属农业行业
   - registration_code: 注册代码
   - established_date: 成立时间
   - registered_capital: 注册资本
   - employees: 员工/社员数量
   - location: 所在地（县乡村）
   - esg_score: ESG评分
   - last_evaluation: 最后评价时间
   - annual_revenue: 年收入
   - carbon_assets: 碳资产数量
   - contact_person: 联系人
   - contact_phone: 联系电话

2. 区块链交易数据
   - id: 交易ID
   - transaction_hash: 交易哈希
   - block_number: 区块号
   - transaction_type: 交易类型
   - from_address: 发送地址
   - to_address: 接收地址
   - amount: 交易金额
   - gas_used: Gas消耗
   - timestamp: 交易时间戳
   - status: 交易状态

3. 碳资产数据
   - id: 资产ID
   - asset_name: 资产名称
   - project_name: 项目名称
   - asset_type: 资产类型（CER/CCER/VER/GCER）
   - amount: 数量
   - current_price: 当前价格
   - total_value: 总价值
   - project_years: 项目年限
   - certification_date: 认证日期
   - expiry_date: 到期日期
   - location: 项目地点
   - methodology: 方法学
   - project_developer: 项目开发商

安装与运行
----------
1. 环境要求
   - Python 3.6 或更高版本
   - 现代浏览器（Chrome、Firefox、Edge等）

   (1) 确保Python环境已安装

   (3) 运行命令：
       python ultra_server.py

AI评分算法
--------
1. 基础评分算法
ESG评分采用加权平均算法：
总体评分 = 环境评分 × 35% + 社会评分 × 30% + 治理评分 × 35%

2. AI增强算法
- 深度学习模型自动调整权重
- 基于历史数据的动态权重优化
- 行业特征自适应调整

3. 农业行业调整因子：
- 种植业: 0.93
- 养殖业: 0.91
- 林业: 0.95
- 农产品加工: 0.89
- 农业科技: 0.94
- 生态农业: 0.97
- 休闲农业: 0.92
- 农机服务: 0.88
- 农资供应: 0.87
- 农村电商: 0.90

评级标准：
- A+：90分及以上（优秀）
- A ：80-89分（良好）
- B+：70-79分（中等偏上）
- B ：60-69分（中等）
- C ：60分以下（需要改进）

农业碳汇类型说明
----------------
1. CER (Certified Emission Reduction)
   - UNFCCC认证的核证减排量
   - 适合大型农业和林业项目
   - 价格较高（约45-65元/吨）

2. CCER (Chinese Certified Emission Reduction)
   - 中国核证减排量
   - 国内农业减排项目首选
   - 价格中等（约35-50元/吨）
   - 适用项目：
     * 保护性耕作
     * 农林复合经营
     * 沼气工程
     * 秸秆还田

3. VER (Verified Emission Reduction)
   - 自愿市场核证减排量
   - 小农户友好型项目
   - 价格较低（约25-35元/吨）
   - 适用项目：
     * 有机农业转换
     * 草畜平衡
     * 农田水利改造

4. GCER (Gold Standard Certified Emission Reduction)
   - 黄金标准认证
   - 高质量可持续农业项目
   - 价格最高（约65-75元/吨）
   - 适用项目：
     * 生态农业示范区
     * 农业生物多样性保护
     * 农村可再生能源
