## 二、软件说明

### 2.1 软件用途

本软件是一款面向银行业的智能金融产品推荐系统，主要应用于银行社保卡客户的精准营销场景。系统基于机器学习算法，通过对客户画像、产品属性和历史行为数据的分析，实现金融产品的智能化、个性化推荐。

**主要应用场景：**
- 银行社保卡客户的产品转化营销
- 客户画像分析与风险评估
- 新产品市场推广与冲突检测
- 客户生命周期管理

### 2.2 技术特点

1. **混合架构设计**：采用Node.js后端 + Python算法引擎的混合架构，兼顾性能和算法灵活性
2. **两步推荐策略**：通过初步推荐和反馈优化的两阶段机制，实现准确率持续提升
3. **不平衡数据处理**：针对银行客户数据中95%仅持有储蓄账户的极不平衡样本，采用SMOTE、ADASYN等先进采样技术
4. **事件驱动分析**：基于客户历史行为事件构建特征，深度挖掘营销机会
5. **产品冲突检测**：新产品上线时自动检测与现有产品的冲突，优化产品布局

### 2.3 技术架构

**前端技术栈：**
- HTML5 + CSS3
- Bootstrap 5.1.3
- Font Awesome 6.0.0
- 原生JavaScript (ES6+)

**后端技术栈：**
- Node.js 18+ + Express.js 5.1.0
- Python 3.8+ (算法引擎)
- scikit-learn (机器学习)
- pandas + numpy (数据处理)
- imbalanced-learn (不平衡数据处理)

**数据规模：**
- 客户数据：110万+条
- 事件数据：33万+条
- 产品数据：259+条
