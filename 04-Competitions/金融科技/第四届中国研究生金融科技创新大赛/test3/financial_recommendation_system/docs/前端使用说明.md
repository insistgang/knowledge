# 智能金融产品推荐系统 - 前端使用说明

## 🎯 系统概述

这是一个完整的网页版金融产品推荐系统前端界面，实现了您要求的所有功能：
- 客户信息查询和展示
- 第一次推荐策略生成
- 用户反馈收集和第二步推荐优化
- 业务事件历史查看
- 营销标签生成
- 完整的用户分析报告

## 📋 功能详解

### 1. 客户信息标签
显示客户的详细基本信息：
- 年龄、性别、教育背景
- 开户日期、客户龄期
- 持有产品数量、成功业务数
- 客户价值分析（高价值、成长潜力、流失风险等）

### 2. 推荐策略标签
**第一次推荐：**
- 显示TOP5推荐产品
- 包含产品名称、描述、推荐理由
- 置信度评分（百分比）
- 高置信度产品会特殊标记

**第二步推荐（基于反馈）：**
- 收集用户对前3个推荐的反馈
- 动态调整推荐策略
- 显示调整原因和置信度变化

### 3. 业务事件标签
- 按时间倒序显示客户所有业务事件
- 区分成功事件（A-开立，B-开通）和普通事件（D-关闭）
- 显示事件详情：产品名称、日期、期限、金额

### 4. 营销标签标签
自动生成客户营销标签：
- **高价值客户**：成功业务超过5次
- **成长潜力**：35岁以下且产品少于3种
- **流失风险**：3个月内无业务活动
- **投资偏好/信贷需求**：基于历史产品分析

### 5. 用户报告标签
生成完整的客户分析报告，包含：
- 客户概览和价值分析
- 产品持有统计分析
- 推荐产品详细分析
- 客户预期分析
- 营销建议和策略

## 🔄 交互流程

### 标准使用流程：
1. **搜索客户** → 输入客户号 → 点击"搜索客户"
2. **查看信息** → 浏览各标签页了解客户情况
3. **收集反馈** → 在左侧选择对推荐产品的反馈
4. **优化推荐** → 点击"提交反馈并重新推荐"
5. **查看结果** → 查看第二步优化推荐
6. **生成报告** → 在"用户报告"标签查看完整分析

### 反馈选项说明：
- **感兴趣**：提升该产品推荐优先级
- **不感兴趣**：降低该产品推荐优先级
- **已有同类**：大幅降低该产品推荐优先级

### 推荐置信度
- **80%以上**：高置信度，蓝色渐变背景
- **60-80%**：中等置信度，橙色渐变背景
- **60%以下**：低置信度，普通背景

### 事件类型标识
- **绿色标签(A)**：开立事件
- **蓝色标签(B)**：开通事件
- **红色标签(D)**：关闭事件

### 营销标签类型
- **橙色**：高价值客户
- **粉色**：潜力客户
- **红色**：风险客户
